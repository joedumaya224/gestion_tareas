# Project Setup

## Requisitos previos

Asegúrate de tener instalados los siguientes programas:

- [Node.js](https://nodejs.org) y npm (Node Package Manager)
- [Composer](https://getcomposer.org)
- [PHP](https://www.php.net)
- [Git](https://git-scm.com)
